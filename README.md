## About
```bash
Loyha olx.uz saytidan e’lonlarni keyword bo‘yicha parsing qilish,
ma’lumotlarni MongoDB Atlas ga saqlash va har bir run’da yangi (insert) va yangilangan (update) e’lonlarni hisoblash uchun mo‘ljallangan .

Loyihada keywords.txt fayliga kalit so‘zlar yoziladi. Har bir qatorda:
   - kalit so‘z
   - ixtiyoriy: [N] — shu kalit so‘z bo‘yicha nechta e’lon parsing qilinishini bildiradi.
```

## Project setup
```bash

$ pip install -r requirements.txt 
$ py -m pip install -r requirements.txt
$ playwright install chromium

```

## Compile and run the project

```bash 
# development
$ python3 main.py

```
